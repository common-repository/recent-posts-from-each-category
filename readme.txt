== Description ==
[Click here to get Pro Version.](http://mindstien.com/pro-plugins/recent-posts-from-each-category-pro.php).

1.3 Updates: Admin interface added

Automatically display recent posts from all/selected categories in box layout on homepage. You can fully configure the layout using shortcode generator. 

Now Use built int shortcode generator to configure all options.

Just put shortcode on any wordpress page/post to display the recent posts from each category. To display it on homepage, you have to configure wordpress homepage as static page and then use the shortcode in content of the static page.

Visit official website for this plugin at [Mindstien Technologies](http://www.mindstien.com).

== Installation ==

1. Install and activate the plugin just like any other wordpress plugin.
1. Then use shortcode as explained in description in your page/post content. this plugin does not have any option page in wp-admin panel.

= How to configure ? =

Visit the plugin options page and make changes to different options available, that will automatically applied to shortcode generator, copy the shortcode and paste any where you want.
